Ударные брызги


Название охуенно

Имеющиес ресурсы:
1) кровь и кишки
2) металлические ошметки
3) костяные куски

По 3 разновидности на вариацию

Крови 4 но 1 юзается под фури.  Фуре уберем

Проявляться эффект будет при критическом попадании НЕ магическим уроном ( у магии итак эффекты свои есть). То есть 1. стрелы и 2. мили.

Инициация: при критическом попадании, с вероятностью Х% (процент я сбалансирую, оценив эффект на практике).

Вариации степени удара:
так как степени фактически 3, то третья будет юзаться только если удар - смертельный (да, и критический одновременно!)
1и2 вариация используются вне зависимости от того смертелен удар или нет.
Шанс на отображение анимации един при проверке на тип (смертельная или нет.

Тип анимации пойдет по монстр-слотам, т.к. прямой зависимости от типа монстра - нет.

Окрас брызгов КРАЙНЕ желательно синхронизировать с TRN цветом монстра. Иначе будет цветовой бардак.
При текущей системе организации это невозможно
Значит надо ограничить тип монстров, для которых цвета брызгов по дефолту будут уместны.
Кровь будет для всех уместна. ну разве что кроме насекомых у которых хз какого цвета кровь
Это тупо по монстр слотам делать надо

Итого при крите:
Шаг 1 проверяем есть ли у данного монстра анимация брызгов
Шаг 2 делаем проверку на смертельность атаки (уже имеется, надо только вклиниться)
Шаг 3 делаем проверку на шанс проигрыша анимации (разный у смертельной и несмертельной)
Шаг 4 запускаем анимацию выбранную по координатам монстра

@critSplashNumbers: - новая табличка из ~500 байт. Каждый байт будет обозначать какую анимацию крит сплеша монстр имеет, если имеет  / A38B00 ( 405700 )

@IsMonsterHasCritSplash: / EA490
; ecx - индекс монстра
; eax - 0 - нет, 1 - кровь MI_17_BLOOD, 2 кость MI_18_BONE, 3 - металл MI_19_METLHIT
imul ecx, 228
mov ecx, dword ptr [69DD64h + ecx] ; monsterSpritePtr
mov ecx, dword ptr [ecx] ; baseMonsterIndex
movzx eax, byte ptr [@critSplashNumbers + ecx]
; место для особенных проверок
retn

@CritAnimationsIndexTable: - новая табличка из 9 байт. Чтобы не городить мега проверки / A38D20

1, 0, 0, ; blood2, bone1, metlhit1-metlhit2,  - lite
1, 0, 1,

@CastMonsterCritSplash: / EA500 ( 71D900 )

; edx - смертельный ли удар

push ebx; targetCol

@heavy:

@@:
