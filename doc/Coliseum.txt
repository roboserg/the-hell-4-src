
Game mode name: Coliseum

Allows all classes to play. 
GCs not allowed. 
Traits - not allowed.
Game difficulties (Horror-Purgatory-Doom) - not allowed.
Game modes (Normal, Easy, Hard, ..) - not allowed.

All game is PvP on (manually pre-made) arenas. (maybe with some PvE elements too - not decided yet).
Opponents are bots or real players. 1 on 1 and 2 on 2 fights.

Question is: will it not be too boring if arena fights are only PvP? What if we make it like Crucible in Grim Dawn, with waves of enemies spawned?
_______________________

It will require, if we go strictly PvP route:
    a. new affix, rare affix and enchant (or will i turn it off there? ) tables, (lots of affixes dont work in PvP, so they have to be removed in Coliseum)
    b. loot is probably absent and will be acquired through vendors
    c. perk and syn tables will need to be different as well, too many PvE choices there.. or maybe even turn off will be required (we'll see, not decided yet)

Fighting goes on like this: player approaches Sepia, opens dialogue window and chooses fight type (1v1 or 2v2)
and selects gladiator type by char.level: 1-5, 6-10, 11-15, 16-20, 21-25, 26-30, 31-35, 36-40, 41-45, 46-50. (and bear in mind that we can increase CLVL cap)

This teleports player to a (random?) arena. And battle starts when all fight members have arrived (or at once, it's a fight with bot(s) and/or with monsters).

The winner gets XP according to foe level. Approximately, CLVLx1000. A portal opens in a corner of arena, player enters it and gets teleported back to town. Cycled loops...
