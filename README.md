This is Readme file to accompany The Hell 4 source.
Dated March 3, 2026.

The mod is almost complete at this moment. My vision of it as end product is mostly achieved. But there is still place for improvements.
This source remained private since The Hell 2 times. Because main programmer was against publishing it (though I was the one to get criticism for that).
I believe in open source and sharing knowledge. I began TH1 mostly because The Dark mod developers shared their workshop for it.
Now that lead programmer is gone, I don't need to abide by this restriction anymore. Hence opensourcing.

You are free to do as you wish with this source. Create derivative works or whatever else. Just give them different name. Not "The Hell".

You can redistribute the source but with this Readme included.

In its core it is Hellfire v1.01 that was decompiled with IDA long time ago (it began in 2007).
It's not completely cleaned out, some of that decompilatus Diablii is still there but it's mostly functions that are not really in the need of edits.

The Hell mod developers used Visual Studio.

How to install the source:
1. InitProject.bat
2. After this, edit personal.props (set the directory for the mod)
3. You may need to install MSVC v141 compile tools in VS too (usually it's "FXC.exe missing" error)

I don't include my workshop along with this source as it is several gigs in size. And I don't want to publish it.
